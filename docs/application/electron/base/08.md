# 八、electron-vite

electron-vite 一站式覆盖，不再需要繁琐的配置

[进入官网](https://electron-vite.github.io/)

## 8.1 简单使用

1. 安装

    ```sh
    npm create electron-vite@latest
    ```

2. 进入安装目录下安装插件

    ```sh
    npm install
    ```

3. 执行命令

    ```sh
    npm run dev
    ```
