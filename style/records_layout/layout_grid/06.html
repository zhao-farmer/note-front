6.1 жҳҫзӨәзҪ‘ж јеұһжҖ§

6.1.1 grid-template-columns/rowsеұһжҖ§

е®№еҷЁжҢҮе®ҡдәҶзҪ‘ж јеёғеұҖд»ҘеҗҺпјҢжҺҘзқҖе°ұиҰҒеҲ’еҲҶиЎҢе’ҢеҲ—гҖӮgrid-template-columnsеұһжҖ§е®ҡд№үжҜҸдёҖеҲ—зҡ„еҲ—е®ҪпјҢgrid-template-rowsеұһжҖ§е®ҡд№үжҜҸдёҖиЎҢзҡ„иЎҢй«ҳгҖӮ

6.1.1.1 з©әеҖј

1. noneпјҡиЎЁзӨәдёҚжҳҺзЎ®зҡ„зҪ‘ж јпјҢжүҖжңүзҡ„иЎҢе’Ңе…¶еӨ§е°ҸйғҪе°Ҷз”ұgrid-auto-rows еұһжҖ§йҡҗејҸзҡ„жҢҮе®ҡгҖӮ

6.1.1.2 ж•°еҖј

1.йқһиҙҹеҖјзҡ„й•ҝеәҰеӨ§е°ҸпјҲpxпјҢemпјҢvwпјү
.container {
    display: grid;
    /*дёӢйқўд»Јз ҒжҢҮе®ҡдәҶдёҖдёӘдёүиЎҢдёүеҲ—зҡ„зҪ‘ж јпјҢеҲ—е®Ҫе’ҢиЎҢй«ҳйғҪжҳҜ100pxгҖӮ*/
    grid-template-columns: 100px 100px 100px;
    grid-template-rows: 100px 100px 100px;
}

е…¶д»–еҸ–еҖјпјҡ

2.йқһиҙҹеҖјдё”зӣёеҜ№дәҺзҪ‘ж је®№еҷЁзҡ„зҷҫеҲҶжҜ”гҖӮ
.container {
    display: grid;
    /*дёӢйқўд»Јз ҒжҢҮе®ҡдәҶдёҖдёӘдёүиЎҢдёүеҲ—зҡ„зҪ‘ж јпјҢеҲ—е®Ҫе’ҢиЎҢй«ҳйғҪжҳҜ100pxгҖӮ*/
    grid-template-columns: 30% 30% 30%;
    grid-template-rows: 30% 30% 30%;
}

6.1.1.3 д»Јжӣҝй•ҝеәҰзҡ„еҚ•дҪҚ

1.frеӯ—з¬ҰдёІ

йқһиҙҹеҖјпјҢз”ЁеҚ•дҪҚ fr жқҘе®ҡд№үзҪ‘ж јиҪЁйҒ“еӨ§е°Ҹзҡ„еј№жҖ§зі»ж•°гҖӮжҜҸдёӘе®ҡд№үдәҶ зҡ„зҪ‘ж јиҪЁйҒ“дјҡжҢүжҜ”дҫӢеҲҶй…Қеү©дҪҷзҡ„еҸҜз”Ёз©әй—ҙгҖӮ

дёәдәҶж–№дҫҝиЎЁзӨәжҜ”дҫӢе…ізі»пјҢзҪ‘ж јеёғеұҖжҸҗдҫӣдәҶfrе…ій”®еӯ—пјҲfraction зҡ„зј©еҶҷпјҢж„Ҹдёә"зүҮж®ө"пјүгҖӮеҰӮжһңдёӨеҲ—зҡ„е®ҪеәҰеҲҶеҲ«дёә1frе’Ң2frпјҢе°ұиЎЁзӨәеҗҺиҖ…жҳҜеүҚиҖ…зҡ„дёӨеҖҚгҖӮ

.container {
    display:grid;
    /*дёӨдёӘзӣёеҗҢе®ҪеәҰзҡ„еҲ—гҖӮ*/
    grid-template-columns: 1fr 1fr;
}

frеҸҜд»ҘдёҺз»қеҜ№й•ҝеәҰзҡ„еҚ•дҪҚз»“еҗҲдҪҝз”ЁпјҢиҝҷж—¶дјҡйқһеёёж–№дҫҝгҖӮ

.container {
  display: grid;
    /*з¬¬дёҖеҲ—зҡ„е®ҪеәҰдёә150еғҸзҙ пјҢз¬¬дәҢеҲ—зҡ„е®ҪеәҰжҳҜз¬¬дёүеҲ—зҡ„дёҖеҚҠгҖӮ*/
  grid-template-columns: 150px 1fr 2fr;
}
2.max-content/min-contentеӯ—з¬ҰдёІ

йҰ–е…Ҳд»Ӣз»ҚдёҖдёӢпјҢmin-contentе’Ңmax-contentе°әеҜёжҳҜж №жҚ®еҶ…е®№жқҘзҡ„пјҢmin-contentжҳҜжңҖе°ҸеҶ…е®№е°әеҜёпјҢдёӯж–Үзҡ„жңҖе°ҸеҶ…е®№еҚ•дҪҚжҳҜдёҖдёӘжұүеӯ—пјҢиӢұж–Үзҡ„жңҖе°ҸеҶ…е®№еҚ•дҪҚжҳҜеҚ•иҜҚпјҢеӣ жӯӨmin-contentжңҖз»Ҳе®ҪеәҰжҳҜжүҖжңүиҝҷдәӣжңҖе°ҸеҶ…е®№еҚ•е…ғжңҖй•ҝзҡ„йӮЈдёӘеҚ•е…ғе®ҪеәҰпјӣmax-contentжҳҜжңҖеӨ§еҶ…е®№е®ҪеәҰпјҢеҸҜд»ҘзҗҶи§Јдёәж–Үжң¬еҶ…е®№дёҚжҚўиЎҢж—¶еҖҷзҡ„е®ҪеәҰгҖӮдёҚиҝҮпјҢmin-contentе’Ңmax-contentеңЁе®һйҷ…ејҖеҸ‘зҡ„ж—¶еҖҷжҳҜдёҚдјҡзӣёеҜ№дәҺеӯ—з¬ҰиҝӣиЎҢе°әеҜёи®ҫе®ҡзҡ„пјҢиҖҢжҳҜзӣёеҜ№дәҺеӣҫзүҮжҲ–иҖ…еҶ…иҒ”жҖ§иҙЁзҡ„е®№еҷЁе…ғзҙ пјҢжҜ”ж–№иҜҙе®№еҷЁе®ҪеәҰдёҚзЎ®е®ҡеҗҢж—¶дёҖиЎҢжңҖеӨҡжҳҫзӨәдёҖдёӘе®№еҷЁпјҲmin-contentпјүпјҢжҲ–иҖ…жүҖжңүе…ғзҙ еңЁдёҖиЎҢжҳҫзӨәпјҲmax-contentпјүгҖӮ

min-content е…ій”®еӯ—еҸҜе°ҶиҪЁйҒ“и®ҫзҪ®дёәдёҺе…¶жңҖе°ҸеҶ…е®№дёҖж ·е®ҪжҲ–дёҖж ·й«ҳгҖӮйҖҡеёёжҳҜеҚ•иҜҚй—ҙжІЎжңүйўқеӨ–з©әй—ҙж—¶зҡ„е°әеҜёгҖӮеҰӮдёӢеӣҫпјҡ

.container {
    display: grid;
    /* ж №жҚ®еҶ…е®№зЎ®и®ӨеӨ§е°Ҹ */
    grid-template-columns: max-content max-content  max-content ;
    
}

max-content е…ій”®еӯ—зҡ„дҪңз”Ёеҹәжң¬дёҠдёҺ min-content зӣёеҸҚпјҡе®ғж №жҚ®зҪ‘ж јеҚ•е…ғж јдёӯжңҖеӨ§зҡ„еҶ…е®№жқҘзЎ®е®ҡиҪЁйҒ“еӨ§е°ҸгҖӮ

.container {
    display: grid;
    /* ж №жҚ®еҶ…е®№зЎ®и®ӨеӨ§е°Ҹ */
    grid-template-columns: min-content min-content  min-content ;
}
3.autoеӯ—з¬ҰдёІ

еҰӮжһңиҜҘзҪ‘ж јиҪЁйҒ“дёәжңҖеӨ§ж—¶пјҢиҜҘеұһжҖ§зӯүеҗҢдәҺ max-content пјҢдёәжңҖе°Ҹж—¶пјҢеҲҷзӯүеҗҢдәҺ min-content гҖӮauto еҸӘжңүеңЁдёҺе…¶д»–еҖјж··еҗҲж—¶жүҚдјҡеҮәзҺ°дёҠиҝ°иЎҢдёәгҖӮ

.container {
    display: grid;
    /* ж №жҚ®еҶ…е®№зЎ®и®ӨеӨ§е°Ҹ */
    grid-template-columns: auto 1fr auto 1fr auto;
}

6.1.1.4 дҪңз”ЁдәҺеҚ•дёӘж•°еҖјеҮҪж•°

1.fit-content()

еҸӘжңүдёҖдёӘеҸӮж•°пјҢеҸӘиғҪдёәй•ҝеәҰжҲ–иҖ…зҷҫеҲҶеҖј гҖӮ

е…¶еә•еұӮеҺҹзҗҶдёҚиҝҮеӨҡи§ЈйҮҠпјҢж•ҲжһңеҸҜд»ҘжҸҸиҝ°дёәпјҡвҖңе°әеҜёз”ұеҶ…е®№еҶіе®ҡпјҢеҶ…е®№и¶ҠеӨҡе°әеҜёи¶ҠеӨ§пјҢжңҖе°Ҹдёә min-contentпјҢжңҖеӨ§дёҚи¶…иҝҮйҷҗе®ҡзҡ„е°әеҜёвҖқгҖӮ

й»ҳи®ӨжҳҜз»ҷзҡ„еҖјпјҢзј©е°ҸеҗҺеҸҜд»ҘдҪҺдәҺи®ҫе®ҡеҖј

.container {
    display: grid;
    /* ж №жҚ®еҶ…е®№зЎ®и®ӨеӨ§е°Ҹ */
    grid-template-columns: fit-content(150px) 100px fit-content(100px) 50px fit-content(100px);
}
2.minmax() еҮҪж•°

minmax() еҮҪж•°жң¬иә«йңҖиҰҒдёӨдёӘеҸӮж•°вҖ“жңҖе°ҸеҖје’ҢжңҖеӨ§еҖјпјҢдёӯй—ҙз”ЁйҖ—еҸ·йҡ”ејҖгҖӮеӣ жӯӨпјҢйҖҡиҝҮ minmax()пјҢжҲ‘д»¬еҸҜд»ҘеңЁзҒөжҙ»зҡ„зҺҜеўғдёӯдёәиҪЁйҒ“и®ҫзҪ®дёҖзі»еҲ—еҸҜиғҪзҡ„е°әеҜёгҖӮ

дҫӢеҰӮпјҢжҲ‘д»¬еҸҜд»Ҙе°ҶдёҖеҲ—и®ҫзҪ®дёә minmax(40px, 100px)пјҢиҝҷж„Ҹе‘ізқҖе…¶жңҖе°Ҹе®ҪеәҰдёә 40pxпјҢжңҖеӨ§е®ҪеәҰдёә 100pxгҖӮminmax() зҡ„дёӨдёӘеҸӮж•°йғҪеҸҜд»ҘдҪҝз”Ёй•ҝеәҰеҖјпјҢеҰӮ frгҖҒpxгҖҒemгҖҒ% е’Ң chпјҢд»ҘеҸҠ min-contentгҖҒmax-content е’Ң autoгҖӮдёҚиҝҮпјҢжңҖеҘҪиҮіе°‘дёәдёҖдёӘеҸӮж•°дҪҝз”Ёй•ҝеәҰеҖјпјҢеӣ дёәе…ій”®еӯ—дёҚеә”иҜҘеҗҢж—¶дҪңдёәдёӨдёӘеҸӮж•°е·ҘдҪң гҖӮ

.container {
    display: grid;
    grid-template-columns:minmax(100px,200px) 120px 80px;
    grid-template-rows: minmax(100px, auto)  70px 130p;
}

minmax() еҮҪж•°зҡ„еҸӮж•°д№ҹеҸҜд»ҘжҳҜ min() жҲ– max() еҮҪж•°гҖӮиҝҷдёӨдёӘеҮҪж•°йғҪжҺҘж”¶дёӨдёӘеҸӮж•°гҖӮmin()еҮҪж•°еә”з”ЁдёӨдёӘеҖјдёӯиҫғе°Ҹзҡ„еҖјпјҢиҖҢ max() еҮҪж•°еә”з”ЁиҫғеӨ§зҡ„еҖјгҖӮиҝҷеңЁе“Қеә”ејҸзҺҜеўғдёӯйқһеёёжңүз”ЁгҖӮ

.container {
    display: grid;
    grid-template-columns:minmax(100px,200px) 120px 80px;
    grid-template-rows: minmax(min(100px, 8vw), 1fr) 70px 130p;
}

6.1.1.5 дҪңз”ЁдәҺж•ҙдҪ“ж•°еҖјеҮҪж•°

1.repeat()еҮҪж•°

repeat() еҮҪж•°иЎЁзӨәиҪЁйҒ“еҲ—иЎЁзҡ„йҮҚеӨҚзүҮж®өпјҢе…Ғи®ёд»Ҙжӣҙзҙ§еҮ‘зҡ„еҪўејҸеҶҷе…ҘеӨ§йҮҸжҳҫзӨәйҮҚеӨҚжЁЎејҸзҡ„еҲ—жҲ–иЎҢгҖӮеҰӮпјҡgrid-template-rows: repeat(3, 2fr 1fr)пјҡйҮҚеӨҚвҖң2fr 1frвҖқиҝҷдёӨдёӘжЁЎејҸдёүж¬ЎпјҢжҖ»е…ұеҲӣе»ә6иЎҢгҖӮ

repeat()еҮҪж•°жңүдёӨдёӘеҸӮж•°пјҡ

з¬¬дёҖдёӘеҸӮж•°еҸҜд»ҘжҳҜд»ҘдёӢдёүз§Қд№ӢдёҖпјҡ

  • ж•°еӯ—пјҲжҜ”еҰӮ1пјҢ2пјҢ3пјү
  • auto-fitе…ій”®еӯ—
  • auto-fillе…ій”®еӯ—

з¬¬дәҢдёӘеҸӮж•°еҸҜйҖүеҖјеҢ…жӢ¬пјҡ

  • й•ҝеәҰеҖјпјҢеҸҜдҪҝз”ЁеҚ•дҪҚеҢ…жӢ¬frгҖҒpxгҖҒemгҖҒ%е’Ңchзӯүзӯү

auto-fit дёҺ auto-fill зҡ„иЎҢдёәжҳҜзӣёдјјзҡ„пјҢеҢәеҲ«еңЁдәҺauto-fit дјҡжҠҠз©әзҡ„еҢҝеҗҚж јеӯҗиҝӣиЎҢжҠҳеҸ еҗҲе№¶пјҢиҖҢиҝҷдёӘеҗҲе№¶зҡ„ 0px еӨ§е°Ҹж јеӯҗеҸҜд»Ҙи®Өдёәе…·жңүеҚ•дёӘж јеӯҗиҪЁйҒ“еӨ§е°Ҹи°ғж•ҙзҡ„еҠҹиғҪпјҢеҜ№дәҶпјҢе…¶дёӨдҫ§зҡ„ж јеӯҗиҝҮйҒ“д№ҹдјҡеҗҲе№¶гҖӮеңЁ auto-fill иҮӘеҠЁеЎ«е……зҡ„ж—¶еҖҷпјҢеҰӮжһң Gridе®№еҷЁзҡ„е°әеҜёзү№еҲ«зҡ„е®ҪпјҢеҲҷжңҖеҗҺдјҡжңүдёҖдәӣз©әзҡ„ж јеӯҗеҚ дҪҚпјҡ

  • пјҡйқһиҙҹзҡ„гҖҒд»ҘfrдёәеҚ•дҪҚзҡ„з»ҙеәҰпјҢжҢҮе®ҡиҪЁйҒ“зҡ„еј№жҖ§еӣ еӯҗгҖӮжҜҸдёӘ е°әеҜёзҡ„иҪЁйҒ“йғҪжҢүе…¶еј№жҖ§еӣ еӯҗзҡ„жҜ”дҫӢжҢӨеҚ еү©дҪҷз©әй—ҙзҡ„дёҖйғЁеҲҶгҖӮ
  • max-content:пјҡе…ій”®иҜҚпјҢжҢҮжҳҺз”ұзҪ‘ж је…ғзҙ дёӯеҚ з”Ёз©әй—ҙжңҖеӨ§зҡ„йӮЈдёҖдёӘжқҘеҶіе®ҡиҪЁйҒ“зҡ„е°әеҜёгҖӮ

6.4 жҺ’еҲ—ж–№ејҸеұһжҖ§

6.4.1 justify-content

justify-contentеұһжҖ§жҳҜж•ҙдёӘеҶ…е®№еҢәеҹҹеңЁе®№еҷЁйҮҢйқўзҡ„ж°ҙе№ідҪҚзҪ®пјҲе·ҰдёӯеҸіпјүгҖӮ

justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
  1. start - еҜ№йҪҗе®№еҷЁзҡ„иө·е§Ӣиҫ№жЎҶ

  1. end - еҜ№йҪҗе®№еҷЁзҡ„з»“жқҹиҫ№жЎҶ

  1. center - е®№еҷЁеҶ…йғЁеұ…дёӯ

  1. stretch - йЎ№зӣ®еӨ§е°ҸжІЎжңүжҢҮе®ҡж—¶пјҢжӢүдјёеҚ жҚ®ж•ҙдёӘзҪ‘ж је®№еҷЁгҖӮ

  1. space-around - жҜҸдёӘйЎ№зӣ®дёӨдҫ§зҡ„й—ҙйҡ”зӣёзӯүгҖӮжүҖд»ҘпјҢйЎ№зӣ®д№Ӣй—ҙзҡ„й—ҙйҡ”жҜ”йЎ№зӣ®дёҺе®№еҷЁиҫ№жЎҶзҡ„й—ҙйҡ”еӨ§дёҖеҖҚгҖӮ

  1. space-between - йЎ№зӣ®дёҺйЎ№зӣ®зҡ„й—ҙйҡ”зӣёзӯүпјҢйЎ№зӣ®дёҺе®№еҷЁиҫ№жЎҶд№Ӣй—ҙжІЎжңүй—ҙйҡ”гҖӮ

  1. space-evenly - йЎ№зӣ®дёҺйЎ№зӣ®зҡ„й—ҙйҡ”зӣёзӯүпјҢйЎ№зӣ®дёҺе®№еҷЁиҫ№жЎҶд№Ӣй—ҙд№ҹжҳҜеҗҢж ·й•ҝеәҰзҡ„й—ҙйҡ”гҖӮ

6.4.2 align-content

align-contentеұһжҖ§жҳҜж•ҙдёӘеҶ…е®№еҢәеҹҹзҡ„еһӮзӣҙдҪҚзҪ®пјҲдёҠдёӯдёӢпјүгҖӮ

align-content: start | end | center | stretch | space-around | space-between | space-evenly;

align-contentеұһжҖ§зҡ„еӣҫе®Ңе…ЁдёҖж ·пјҢеҸӘжҳҜе°Ҷж°ҙе№іж–№еҗ‘ж”№жҲҗеһӮзӣҙж–№еҗ‘гҖӮ
